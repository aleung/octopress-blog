如果你习惯使用UNIX的命令，那么在Windows中...

2004-02-20 | Software

unxutils.sourceforge.net 提供了一系列的unix工具在win32平台上的移植版本. 这些版本是完全基于win32平台的, 不像cygwin那样还需要一个模拟层.

Windows 2000 command line 中的 for 命令用于对一组文件中的每个执行某特定命令，类似于unix的 find * -exec command ，功能很强大，具体参数用 for /? 查看。
